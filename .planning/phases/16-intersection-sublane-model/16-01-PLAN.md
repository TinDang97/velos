  <name>Task 1: BezierTurn, ConflictPoint, and junction precomputation</name>
  <files>crates/velos-net/src/junction.rs, crates/velos-net/src/lib.rs</files>
  <behavior>
    - BezierTurn::position(0.0) == p0, BezierTurn::position(1.0) == p2
    - BezierTurn::tangent(0.0) is proportional to (p1 - p0)
    - BezierTurn::tangent(1.0) is proportional to (p2 - p1)
    - BezierTurn::offset_position shifts perpendicular to tangent (left-pointing normal)
    - estimate_arc_length returns > 0 for non-degenerate curves
    - find_conflict_point returns Some for two crossing straight-ish paths, None for parallel paths
    - precompute_junction generates turns for each (incoming, outgoing) pair, skipping U-turns
    - precompute_all_junctions produces a HashMap<u32, JunctionData> covering all junction nodes
  </behavior>
  <action>
Create `crates/velos-net/src/junction.rs` with:

1. `BezierTurn` struct per RESEARCH.md Pattern 1:
   - Fields: `entry_edge: u32`, `exit_edge: u32`, `p0: [f64; 2]`, `p1: [f64; 2]`, `p2: [f64; 2]`, `arc_length: f64`
   - `fn position(&self, t: f64) -> [f64; 2]` -- quadratic Bezier evaluation
   - `fn tangent(&self, t: f64) -> [f64; 2]` -- B'(t) = 2(1-t)(P1-P0) + 2t(P2-P1)
   - `fn offset_position(&self, t: f64, lateral_offset: f64, road_half_width: f64) -> [f64; 2]` -- shift perpendicular to tangent
   - Control points: P0 = source node pos of entry edge, P1 = junction node pos (centroid), P2 = target node pos of exit edge

2. `ConflictPoint` struct per RESEARCH.md Pattern 2:
   - Fields: `turn_a_idx: u16`, `turn_b_idx: u16`, `t_a: f32`, `t_b: f32`

3. `JunctionData` struct:
   - Fields: `turns: Vec<BezierTurn>`, `conflicts: Vec<ConflictPoint>`

4. `fn estimate_arc_length(p0, p1, p2, steps: usize) -> f64` -- sample N points, sum segment lengths

5. `fn find_conflict_point(a: &BezierTurn, b: &BezierTurn, steps: usize) -> Option<(f32, f32)>` -- grid search, 2m threshold

6. `fn precompute_junction(graph, node) -> JunctionData` -- iterate incoming x outgoing edges, skip U-turns, build BezierTurns, find ConflictPoints between all turn pairs

7. `fn precompute_all_junctions(graph: &RoadGraph) -> HashMap<u32, JunctionData>` -- iterate all nodes with degree >= 2 (both in and out edges), call precompute_junction for each

Add `pub mod junction;` to `crates/velos-net/src/lib.rs` and re-export key types.

IMPORTANT: Use `graph[node].pos` for Bezier control points (actual node positions), NOT the last/first point of edge geometry polylines. This avoids coordinate mismatch at junction entry/exit (see RESEARCH.md Pitfall 1).

Arc length estimation: use 20 sample points (sufficient for typical 10-30m junction curves per RESEARCH.md discretion).

```rust
/// ECS component for an agent traversing a junction on a Bezier curve.
/// Attached when agent enters junction, removed when agent exits to next edge.
#[derive(Debug, Clone, Copy, PartialEq)]
pub struct JunctionTraversal {
    /// Index of the junction node in the road graph.
    pub junction_node: u32,
    /// Index of the BezierTurn in the junction's turns array.
    pub turn_index: u16,
    /// Current parameter along the Bezier curve [0.0, 1.0].
    pub t: f64,
    /// Locked lateral offset (from when agent entered junction).
    pub lateral_offset: f64,
    /// Speed along the curve (m/s).
    pub speed: f64,
}
```
