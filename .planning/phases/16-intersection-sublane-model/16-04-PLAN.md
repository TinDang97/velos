**A) Create `crates/velos-gpu/shaders/guide_line.wgsl`:**

Dashed line shader for Bezier guide lines through junctions. Use the same camera uniform as map_tile.wgsl. The dash pattern can be achieved by discarding fragments based on distance along the line:

```wgsl
struct CameraUniform { view_proj: mat4x4f };
@group(0) @binding(0) var<uniform> camera: CameraUniform;

struct VertexInput {
    @location(0) position: vec2f,
    @location(1) color: vec4f,
    @location(2) line_dist: f32,
};

struct VertexOutput {
    @builtin(position) clip_position: vec4f,
    @location(0) color: vec4f,
    @location(1) line_dist: f32,
};

@vertex
fn vs_main(in: VertexInput) -> VertexOutput { ... }

@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4f {
    let pattern = fract(in.line_dist / 5.0);
    if (pattern > 0.6) { discard; }
    return in.color;
}
```

**B) Add guide line and debug rendering to Renderer:**

Add to `crates/velos-gpu/src/renderer.rs`:

1. `guide_line_pipeline: wgpu::RenderPipeline` -- created from guide_line.wgsl
2. `guide_line_buffer: Option<wgpu::Buffer>` and `guide_line_vertex_count: u32`
3. `debug_overlay_buffer: Option<wgpu::Buffer>` and `debug_overlay_vertex_count: u32`

4. `pub fn update_guide_lines(&mut self, device: &wgpu::Device, junction_data: &HashMap<u32, JunctionData>)`:
   - For each junction, for each BezierTurn: sample ~20 points along the curve
   - Generate thin line quads (2 vertices per segment, ~0.5m width) with line_dist attribute
   - Color: semi-transparent white [1.0, 1.0, 1.0, 0.3]
   - Upload to guide_line_buffer

5. `pub fn update_debug_overlay(&mut self, device: &wgpu::Device, junction_data: &HashMap<u32, JunctionData>, active_conflicts: &[(u32, u16, u16)])`:
   - Conflict crossing points: for each ConflictPoint, evaluate both turns at conflict t to get world position, render as red dot (small quad)
   - Active conflict pair lines: for each active_conflicts entry, draw line between the two agents' positions
   - Upload to debug_overlay_buffer

6. In `render_frame()`: after agent pass, conditionally render guide lines and debug overlay based on `show_guide_lines: bool` and `show_conflict_debug: bool` flags.
