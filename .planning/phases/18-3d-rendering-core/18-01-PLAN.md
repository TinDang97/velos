Create `crates/velos-gpu/src/orbit_camera.rs` with:

1. `ViewMode` enum: `TopDown2D`, `Perspective3D`.

2. `ViewTransition` struct: `from: ViewMode`, `to: ViewMode`, `progress: f32` (0.0-1.0), `elapsed: f32`. Method `tick(dt: f32) -> bool` advances elapsed, returns true when complete. Duration constant 0.5s.

3. `OrbitCamera` struct per RESEARCH.md Pattern 2:
   - Fields: `focus: Vec3`, `distance: f32`, `yaw: f32`, `pitch: f32`, `fov_y: f32`, `near: f32`, `far: f32`, `viewport: Vec2`
   - `new(viewport: Vec2)` -- defaults: focus=(0,0,0), distance=500.0, yaw=0.0, pitch=45deg, fov_y=45deg, near=0.1, far=10000.0
   - `eye_position(&self) -> Vec3` -- spherical to Cartesian: x=dist*cos(pitch)*cos(yaw), y=dist*sin(pitch), z=dist*cos(pitch)*sin(yaw), offset by focus
   - `view_proj_matrix(&self) -> Mat4` -- glam::Mat4::look_at_rh(eye, focus, Vec3::Y) * perspective_rh(fov_y, aspect, near, far)
   - `resize(&mut self, new_viewport: Vec2)`
   - `orbit(&mut self, dyaw: f32, dpitch: f32)` -- adds to yaw/pitch, clamps pitch to [5deg, 89deg] in radians
   - `zoom_by(&mut self, factor: f32)` -- multiplies distance, clamp to [1.0, 50000.0]
   - `pan(&mut self, dx: f32, dy: f32)` -- moves focus in camera-right and camera-forward-on-ground directions
   - `from_camera_2d(cam: &Camera2D) -> Self` -- maps center to focus (x, 0, y), zoom to distance (inverse relationship), pitch=45deg, yaw=0

4. `create_depth_texture(device, width, height) -> TextureView` -- Depth32Float, RENDER_ATTACHMENT usage.

5. GPU instance types for 3D (used by later plans):
   - `MeshInstance3D`: `[f32;3] world_pos, f32 heading, [f32;4] color` (32 bytes)
   - `BillboardInstance3D`: `[f32;3] world_pos, [f32;2] size, [f32;4] color, [f32;1] _pad` (40 bytes)
   - Both derive `Copy, Clone, Pod, Zeroable` with `#[repr(C)]`

6. LOD constants: `LOD_MESH_THRESHOLD: f32 = 50.0`, `LOD_BILLBOARD_THRESHOLD: f32 = 200.0`, `HYSTERESIS_FACTOR: f32 = 1.1`.

Coordinate convention: 2D (x, y) maps to 3D (x, 0, y) -- Y is up. Document this prominently.
